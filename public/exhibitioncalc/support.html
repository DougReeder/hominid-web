Installation Problems

If you encounter problems downloading or installing ExhibitionCalc, please search for a similar problem on the WebOS Nation forums.

Contacting Us

If you're experiencing a problem using ExhibitionCalc:

  1. From the Launcher, launch ExhibitionCalc.
  2. Tap the Help button
  3. Scroll to the bottom and tap Send E-Mail

This will create a new e-mail, addressed to us, where you can describe the problem.
