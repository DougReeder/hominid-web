Places
Some Tasks can only be done at home, not at work or while running errands (and vice-versa). To set the Place for a Task, tap the Task to select it, tap the Details button, and select from the Place pop-up.

Due Date, Lead Time & Repeat
To set a deadline for a Task, or change an ordinary Note into an appointment:
  1. tap the Item to select it
  2. tap the Details button
  3. set the Due Date toggle button to ON
  4. set the date selector
Notes and incomplete Tasks appear in the webOS Calendar application on their Due Date. You can can set some or all overdue Tasks to display on today in the webOS Calendar, by setting OVERDUE TASKS in the Project Settings.
If a Task has Lead Time set to “Unlimited” (the default), it will immediately appear in the Task Scenes. If Lead Time is set to a smaller value, it will not appear in a Task Scene until that interval before the Due Date. Example: If the Due Date is set to May 10, and the Lead Time is “3 days”, the Task will first appear in a Task Scene on May 7.
Lead Time keeps the To-Do scene free of Tasks you can't work on yet.
Some Tasks repeat on a fixed schedule. For example, if your rent is due on the 1st of the month, the June rent is due on June 1, whether your May rent was paid early or late. If you're out of the country for three months, you'll owe three months rent when you return. Set these Tasks to have an Exact Repeat (one without “Lax” in the name).
Other Tasks need to be done at intervals, but the interval can vary. For example, if you've been vacuuming your carpet weekly on Mondays, but one week you don't get to it until Friday, you don't need to vacuum it again until the following Friday. If you go on vacation for three weeks, you don't need to vacuum the carpet three times when you get home! Set these Tasks to have a Lax Repeat. Typically, you'll set the period longer — to vacuum “weekly” (meaning every 6–8 days), set a Custom Lax Daily Repeat every 8 days with a Lead Time of 2 days.
