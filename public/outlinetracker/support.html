Installation Problems

If you encounter problems downloading or installing Outline Tracker, please search for a similar problem on the WebOS Nation forums.

Contacting Us

If you're experiencing a problem using Outline Tracker:

  1. From the application menu, select Help
  2. Tap Send E-Mail

This will create a new e-mail, addressed to us, where you can describe the problem. Recent status messages will automatically be included, helping us solve your problem.
