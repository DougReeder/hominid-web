How do the “M+”, “MR” and “MC” keys work?

  • M+ (memory plus) adds the current result to the value in memory.
  • MR (memory recall) replaces the current result with the value in memory.
  • MC (memory clear) clears the memory.

How do I repeat an operation?

Tapping the = key again repeats the last operation.
