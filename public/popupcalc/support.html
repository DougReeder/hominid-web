Installation Problems

If you encounter problems downloading or installing PopupCalc, please search for a similar problem on the WebOS Nation forums.

Contacting Us

If you're experiencing a problem using PopupCalc:

  1. From the Launcher, launch PopupCalc.
  2. From the application menu, select Help
  3. Scroll to the bottom and tap Send E-Mail

This will create a new e-mail, addressed to us, where you can describe the problem.
